\doxysubsection{Detailed Description}
Класс для представления блюда с добавками 

Класс dish содержит основную информацию о блюде и трех добавках. Позволяет рассчитать общую стоимость блюда с учетом всех компонентов.

Общая стоимость рассчитывается по формуле:\+   \[C_{total} = W \times C_{base} + W \times \sum_{i=1}^{3} C_{spice_i}
\]
