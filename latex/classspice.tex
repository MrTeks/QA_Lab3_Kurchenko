\doxysubsection{Detailed Description}
Класс для представления специи/\+добавки к блюду 

Класс spice хранит информацию о стоимости специи и её количестве. Позволяет рассчитать общую стоимость специи для блюда.

Формула расчета стоимости специи:\+   \[C_{spice} = p \times n
\] где:\+
\begin{DoxyItemize}
\item p -\/ стоимость за грамм
\item n -\/ количество грамм на кг 
\end{DoxyItemize}
